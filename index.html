📐 Generate Grid — Choose Method
Option A — 3-Point Measurement
Physically record 3 stage positions: origin (0,0), hole (0,1), and hole (1,0).
Calculates row pitch, column pitch, and stagger from real measurements.
Best for: calibration of new MLA, captures real geometry.
Time: ~10 minutes (3 alignments).
Option B — From Pixel File + Scale Factor
Load existing pixel coordinate file and apply scale factor (3.45 µm/px).
Record only the origin (0,0). All 144 positions calculated from file.
Formula: d = sqrt(dx² + dy²) × 3.45 µm
Best for: reuse of image-processed centroid data.
Time: ~3 minutes (1 alignment).
